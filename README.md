# Cloud and SQL Learning Repository

A comprehensive learning resource covering cloud computing fundamentals and SQL database concepts. This repository contains detailed documentation on cloud platforms (AWS, Azure, GCP) and SQL practices.

## 📚 Repository Overview

This repository is organized into two main sections:

1. **Cloud** - Cloud computing concepts, services, and platform-specific documentation
2. **SQL** - SQL fundamentals and practical examples

---

  - Storage Services (Blob, File, Disk Storage, Databases)
